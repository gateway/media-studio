# Keyboard And Mouse Controls

Media Studio shortcuts are ignored while you are typing in a field or when a blocking dialog is open.

## Studio

### Keyboard

| Input | Action |
| --- | --- |
| `G` | Open Projects |
| `N` | Open Graph Studio |
| `P` | Open Presets |
| `S` | Open Settings |
| `I` | Open Reference Library |
| `ArrowLeft` / `ArrowRight` | Move through selected gallery assets |
| `Escape` | Close the asset inspector or lightbox |
| `ArrowUp` / `ArrowDown` | Move through prompt reference suggestions |
| `Enter` / `Tab` | Insert the selected prompt reference suggestion |

### Mouse

| Input | Action |
| --- | --- |
| Click a gallery item | Open the asset inspector |
| Click a favorite icon | Favorite or unfavorite the asset without opening it |
| Click Projects, Presets, Reference library, or Settings | Open the matching Studio panel |
| Click or drag onto a thumbnail drop area | Choose or upload a thumbnail image |

## Graph Studio

Graph Studio shortcuts are ignored while you are typing in a node field.

### Keyboard

| Input | Action |
| --- | --- |
| `Space` | Open node search |
| `C` | Toggle the bottom console |
| `Cmd/Ctrl+Z` | Undo |
| `Cmd/Ctrl+Shift+Z` / `Ctrl+Y` | Redo |
| `Cmd/Ctrl+C` | Copy selected nodes |
| `Cmd/Ctrl+V` | Paste copied nodes |
| `Cmd/Ctrl+M` | Mute selected nodes |
| `Escape` | Close search, menus, previews, side panels, or rename mode |
| `ArrowLeft` / `ArrowRight` | Move through media preview overlay items |

### Mouse

| Input | Action |
| --- | --- |
| Click a node | Select it |
| Shift-click or Cmd/Ctrl-click a node | Add or remove it from the current selection |
| Drag a node | Move it |
| Drag from an output port to an input port | Create a wire |
| Drag from an occupied single-value input | Detach and reroute that wire |
| Right-click empty canvas | Open node search at the pointer |
| Right-click a selected node | Open the node context menu for the selection |
| Cmd/Ctrl-left-drag on empty canvas | Draw a selection box |
| Drag a group title | Move the group and its member nodes |
| Right-click a group title | Open group actions such as enable or mute |
| Mouse wheel / trackpad scroll | Pan or scroll the graph surface, depending on focus |
