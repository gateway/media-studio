# Media Studio Content Library

This folder is for shareable Media Studio content that can live in the repo without depending on a local database.

Use it for curated exports that users can import, review, or submit back as contributions:

- `media-presets/`: portable Media Preset bundles.
- `prompt-recipes/`: portable Prompt Recipe bundles.
- `graph-workflows/`: Graph Studio workflow exports or workflow bundles.

Do not store generated media outputs, local database rows, API keys, private reference images, or machine-specific paths here.

## File Types

- Media Presets: `.media-studio-preset.zip`
- Prompt Recipes: `.media-studio-prompt-recipe.zip`
- Graph workflows: `.media-studio-graph.json`
- Graph workflow bundles with reference media: `.media-studio-graph.zip`

Prefer one folder per reusable idea when a bundle needs notes, screenshots, or multiple related exports.
